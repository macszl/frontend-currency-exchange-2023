**SPA** (Single Page Application) presenting exchange rates. App features:

1. Table of type A of average exchange rates of foreign currencies. (show exchange rate publication date)

4. Currency converter. Can easily go to the currency history page.
5. Popular cryptocurrency coin tracker. (lists name, symbol, price in usd, growth percentage).
